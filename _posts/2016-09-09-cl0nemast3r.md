Cl0neMast3r es un script de Python que fue codificado para facilitarle la vida.

  
Ahora puede elegir fácilmente sus herramientas favoritas de GitHub e instalarlas en su sistema con un solo clic. Aún mejor, puede asegurarse de tener la última versión de sus herramientas favoritas. Todo esto y más lo puedes hacer con Clone Master.  
  
Clone Master se hizo principalmente para pentesters y cazadores de bugs.

  

  

Cl0neMast3r le ofrece muchas opciones:

  
**Descripción de la opción**

  
A   Agregar una herramienta de GitHub usando URL  
F   Encuentra una herramienta en GitHub  
R   Reinstala tus herramientas  
U   Actualiza tus herramientas  
S   Muestra información sobre tus herramientas favoritas  
D   Eliminar la lista de herramientas  
M   Importa tus herramientas favoritas  
X   Exporta tus herramientas a HTML  
 

**Requisitos:**     Python 2.7. \*  
     Request Lo puedes encontrar [aquí](https://pypi.python.org/pypi/requests)  
     Cómo instalar:  
`pip install requests`  
     BeautifulSoup4 Lo puedes encontrar [aquí](https://pypi.python.org/pypi/beautifulsoup4)  
     Cómo instalar:  
`pip install beautifulsoup4`

  

**Instalación:**

  

git clone https://github.com/Abdulraheem30042/Cl0neMast3r.git  
cd Cl0neMast3r /  
pip install -r required.txt
