In this video, we will learn, how to install and mod Google Camera Port on any smartphone using Magisk Module i.e. Google API 2 Enabler and Build.prop editor. We will also learn, how to place config files.  
  
Step 1 - Check if Google2API is working with Camera2 API Probe - [https://play.google.com/store/apps/details?id=com.airbeat.device.inspector&hl=en\_IN](https://play.google.com/store/apps/details?id=com.airbeat.device.inspector&hl=en_IN)  
Step 2 - Check if Your phone is on the list (List is After step 3)  
Step 3.1.1 Edit Build.prop file - Download - [https://play.google.com/store/apps/details?id=com.jrummy.apps.build.prop.editor&hl=en\_IN](https://play.google.com/store/apps/details?id=com.jrummy.apps.build.prop.editor&hl=en_IN)  
Step 3.1.2 - Add line in Build.prop - persist.camera.HAL3.enabled=1  
Step 3.2 Magisk Camera2 API enabler Module for root - [https://forum.xda-developers.com/apps/magisk/module-camera2api-enabler-t3656651](https://forum.xda-developers.com/apps/magisk/module-camera2api-enabler-t3656651)
