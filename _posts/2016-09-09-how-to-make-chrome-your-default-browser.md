Here’s how you can make Chrome your device’s default browser.

In this guide, we will show you how to set Chrome as the default web browser on Windows, macOS, Android, and iOS.

**Note: **To set Chrome as your default browser, you must download and install it first. Windows and macOS can download it from [Google’s website](https://www.google.ca/chrome/) while [iOS](https://apps.apple.com/us/app/google-chrome/id535886823) and [Android](https://play.google.com/store/apps/details?id=com.android.chrome&hl=en_US) use its respected app stores.

Make Google Chrome the Default Browser on Windows
-------------------------------------------------

Open up System settings by pressing Windows key+I, and then click on “Apps.”

![Open the Settings app and click on ](https://www.howtogeek.com/wp-content/uploads/2019/09/2019-09-19_10h08_36.png)

From the pane on the left side, click “Default Apps.”

![Click ](https://www.howtogeek.com/wp-content/uploads/2019/09/2019-09-19_10h09_43.png)

Locate the Web Browser section, click on your current default browser, and then scroll through the list and select “Google Chrome.”
