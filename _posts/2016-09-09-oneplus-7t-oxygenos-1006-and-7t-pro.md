![](https://images.fonearena.com/blog/wp-content/uploads/2019/10/OnePlus-7T-Pro_fonearena-4-1024x640.jpg) OnePlus has released new software updates for the OnePlus 7T and the OnePlus 7T Pro that have small incremental changes with some bug fixes. The 7T and the 7T pro came pre-loaded with Android 10 at launch and have continued to get frequent updates to improve the software experience. The new updates take the OxygenOS version to 10.0.4 on the OnePlus 7T Pro, the update file size being 166MB and version 10.0.6 on the OnePlus 7T, the update file size being 75MB. As usual, this OTA is an incremental one, initially rolling out to a small number of users today with wider rolling out coming in the next few days. The following is the changelog for the OnePlus 7T Pro: Changelog System Optimized standby power consumption Improved system stability and general bug fixes Updated Android security patch to 2019.10 Fixed the black bar issue while charging or playing a video Optimized the Bluetooth connectivity in automobiles Optimized the expanded screenshot feature The following is the changelog for the OnePlus 7T Changelog System Optimized standby power consumption Improved system stability and general bug fixes Optimized the Bluetooth connectivity in automobiles Optimized the expanded screenshot feature Source 1,2
