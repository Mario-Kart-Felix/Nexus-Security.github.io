![](https://images.fonearena.com/blog/wp-content/uploads/2019/09/OnePlus-7T_fonearena-01-1024x639.jpg) OnePlus has pushed another incremental update to the OnePlus 7T, bringing the version number to 10.0.7. The update brings certain bug fixes, improvements to RAM management and updates the Android security patch to 2019.11. Some users have reported that the version number is incorrectly showing the version number as 10.3.0. OnePlus has acknowledged the issue and will return with information as soon as possible The update size is reported to be at 296MB and will be rolled out to everyone through OTA. As usual, this OTA is an incremental one, initially rolling out to a small number of users today with wider rolling out coming in the next few days. Check out the complete changelog below: Changelog System Improved the launching speed of some apps Optimized the RAM Management Optimized the black and white screen issues with some apps Improved system stability and general bug fixes Updated Android security patch to 2019.11 Camera Improved the photo quality Source
