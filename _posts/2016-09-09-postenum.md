**USE**

```

./postenum.sh -s  
./postenum.sh -c
```

  
**Options :**

```

-s : Filesystem [SUID, SGID, Config/DB files, etc.]

-c : The most interesting files  
-n : Network settings  
-p : Services and cron jobs

-t : Fstab credentials and databases checker
```
