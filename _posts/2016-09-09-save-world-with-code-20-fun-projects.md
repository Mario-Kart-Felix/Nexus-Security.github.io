[Lorraine Underwood](https://twitter.com/LMcUnderwood)‘s book [Save the World with Code: 20 Fun Projects for All Ages Using Raspberry Pi, micro:bit, and Circuit Playground Express](https://www.amazon.com/Save-World-Code-Raspberry-Playground/dp/1260457591) has been listed by major booksellers as coming out next month (in the US) from TAB!!

See [Lorraine’s tweet](https://twitter.com/LMcUnderwood/status/1223665282209734656) and [Amazon.com](https://www.amazon.com/Save-World-Code-Raspberry-Playground/dp/1260457591) for more information on preordering.
