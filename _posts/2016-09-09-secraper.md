**SEcraper - Search Engine Scraper Tool With BASH Script.**

  
Search engine [scraper](https://www.kitploit.com/search/label/Scraper "scraper") tool with BASH script.  
  
**Dependency**

*   curl (cli)

  
**Available search engine**

*   [Ask.com](http://ask.com/)
*   [Search.yahoo.com](http://search.yahoo.com/)
*   [Bing.com](http://bing.com/)

  

[](https://www.blogger.com/u/7/null)  
**Installation**

```
git clone [https://github.com/zerobyte-id/SEcraper.git](https://github.com/zerobyte-id/SEcraper.git)  
cd SEcraper/
```

  
**Run**

```
bash secraper.bash "QUERY"
```
