There are multiple vulnerabilities in IBM® Runtime Environment Java![™](https://s.w.org/images/core/emoji/11/72x72/2122.png) Version 7 or Version 8 used by Financial Transaction Manager for Check Services for Multi-Platform (FMT CHK). Financial Transaction Manager for Check Services for Multi-Platform has addressed the applicable CVEs.

Affected Products and Versions
------------------------------

FTM CHK: v3.0.0.0 – 3.0.0.15, v3.0.2.0 – 3.0.2.1, v3.0.5.0 – 3.0.5.4
