There are multiple vulnerabilities in IBM® Runtime Environment Java™ Version 8.0 used by IBM WIoTP MessageGateway. These issues were disclosed as part of the IBM Java SDK updates in October 2019.

**Affected product(s) and affected version(s):**

Affected Product(s)

Version(s)

IBM WIoTP MessageGateway

5.0.0.1

IBM IoT MessageSight

5.0.0.0

2.0
