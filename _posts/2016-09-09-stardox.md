**Stardox - Github Stargazers Information Gathering Tool**

  
Stardox is an advanced github [stargazers](https://www.kitploit.com/search/label/Stargazers "stargazers") [information gathering](https://www.kitploit.com/search/label/Information%20Gathering "information gathering") tool. It scraps Github for information and display them in list tree view.It can be used for collecting information of your's/someones repository stargazers details.  
  
**What data it fetchs :**  

1.  `Total repsitories`
2.  `Total stars`
3.  `Total Followers`
4.  `Total Following`
5.  `Stargazer's Email`

**Getting Started**  
  
**Steps to setup :**  

1.  `git clone [https://github.com/0xprateek/stardox](https://github.com/0xprateek/stardox)`
2.  `cd stardox`
3.  `pip install -r requirements.txt`

  
**Starting [Stardox](https://www.kitploit.com/search/label/Stardox "Stardox") :**  

1.  `cd stardox/src`
2.  a) **Using Command line arguments**   
    `python3 stardox.py [https://github.com/Username/repository-URL](https://github.com/Username/repository-URL) `  
    b) **Without Command line arguments**  
    `python3 stardox.py`

  
**Usage :**  
```
 stardox.py [-h] [-v] repositoryURL
```  
**positional arguments:**  
```
 repositoryURL  Path to repository.
```  
**optional arguments:**  
```
 -h, --help     show this help message and exit  
 -v, --verbose  Verbose
```
