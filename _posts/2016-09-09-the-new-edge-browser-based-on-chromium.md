Today [Microsoft announced](https://blogs.windows.com/windowsexperience/2020/01/15/new-year-new-browser-the-new-microsoft-edge-is-out-of-preview-and-now-available-for-download/) the availability of their Edge web browser (Windows, macOS, Android and iOS) based on the open source Chromium engine. This is a total browser rebuild from previous versions.

Features being touted over other browsers like Google Chrome:

*   Better privacy â€“ tracking prevention is on by default
*   Customizable interface
*   Imports Chrome bookmarks/bookmark bar in one step
*   AAD support
*   Internet Explorer mode
*   4K streaming, Dolby audio
*   Inking in PDF
*   Microsoft Search in Bing integration
*   Support for Chrome-based extensions

The release is for everyone but is being aimed more at organizational users along with Bing integration.
