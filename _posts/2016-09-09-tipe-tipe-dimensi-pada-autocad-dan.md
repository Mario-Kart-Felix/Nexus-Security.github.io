Tipe tipe dimensi pada AutoCAD dan Fungsi dari tools dimensi 

  
Dimensi type Linear ( **DIMLINEAR )** : Dimensi yang ditujukan untuk penggunaan datar / lurus  
  
Dimensi type Aligned  **( DIMALIGNED )**: Dimensi yang penggunaan nya untuk sisi miring bertujuan untuk mengetahui panjang sisi miring / diagonal objek  
  
Dimensi type Angular **(DIMANGULAR)** :untuk mengetahui sudut dari 2 objek yang besinggungan  
  
Dimensi type Arc Length  **(DIMARC)** : bertujuan untuk mengetahui panjang / ukuran dari objek arc  
  
Dimensi type Radius **(DIMRADIUS)** : bertujuan untuk mengetahui radius objek  
  
Dimensi type Diameter **(DIMDIAMETER)** :bertujuan untuk melampirkan diameter objek  
  
Dimensi type jogged **(DIMJOGGED)** : Bertujuan untuk melampirkan dimensi radius pada posisi yang abnormal
