Unfollower adalah tool untuk unfollow secara otomatis untuk pengguna instagram, untuk menggunakan tool tersebut anda membutuhkan kresidesial akun instagram anda  
  
**Installation Unfollower Tool**  
  
Untuk mendapatkan tool tersebut, anda harus melakukan cloning repository berikut  
  
```
git clone https://github.com/thelinuxchoice/unfollower
```

[![](https://1.bp.blogspot.com/-O9a3I_7p4kU/XaCAdUmPE7I/AAAAAAAACpQ/iR7xBiXpf0c1GXD61egvzkqj_jOzdU6FwCEwYBhgL/s320/unfollower-git-www.codecrime.net.png)](https://1.bp.blogspot.com/-O9a3I_7p4kU/XaCAdUmPE7I/AAAAAAAACpQ/iR7xBiXpf0c1GXD61egvzkqj_jOzdU6FwCEwYBhgL/s1600/unfollower-git-www.codecrime.net.png)

  
**Running Unfollower Tool**  
  
Jika sudah melakukan cloning, sekarang ada masuk kedalam directory tool tersebut dan jalankan peritah ini.  
  

[![](https://1.bp.blogspot.com/-GgBBpU_2I3Q/XaCAfRdmNMI/AAAAAAAACpc/1SLPovMB48slm8298fJ0fOXiMLf7X36BQCEwYBhgL/s320/unfollower-run-www.codecrime.net.png)](https://1.bp.blogspot.com/-GgBBpU_2I3Q/XaCAfRdmNMI/AAAAAAAACpc/1SLPovMB48slm8298fJ0fOXiMLf7X36BQCEwYBhgL/s1600/unfollower-run-www.codecrime.net.png)
