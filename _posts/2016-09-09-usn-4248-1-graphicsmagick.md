graphicsmagick vulnerabilities
------------------------------

A security issue affects these releases of Ubuntu and its derivatives:

*   Ubuntu 16.04 LTS

### Summary

Several security issues were fixed in GraphicsMagick.

### Software Description

*   graphicsmagick - collection of image processing tools

### Details

It was discovered that GraphicsMagick incorrectly handled certain image files. An attacker could possibly use this issue to cause a denial of service or other unspecified impact.

Update instructions
-------------------

The problem can be corrected by updating your system to the following package versions:

Ubuntu 16.04 LTS

[graphicsmagick](https://launchpad.net/ubuntu/+source/graphicsmagick) - [1.3.23-1ubuntu0.5](https://launchpad.net/ubuntu/+source/graphicsmagick/1.3.23-1ubuntu0.5)

[libgraphicsmagick++-q16-12](https://launchpad.net/ubuntu/+source/graphicsmagick) - [1.3.23-1ubuntu0.5](https://launchpad.net/ubuntu/+source/graphicsmagick/1.3.23-1ubuntu0.5)

[libgraphicsmagick-q16-3](https://launchpad.net/ubuntu/+source/graphicsmagick) - [1.3.23-1ubuntu0.5](https://launchpad.net/ubuntu/+source/graphicsmagick/1.3.23-1ubuntu0.5)

To update your system, please follow these instructions: [https://wiki.ubuntu.com/Security/Upgrades](https://wiki.ubuntu.com/Security/Upgrades).

In general, a standard system update will make all the necessary changes.
