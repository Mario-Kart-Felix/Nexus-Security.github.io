wpDiscuz Ads Manager - v1.0.5
=============================

  
  
![](https://nullclub.com/wp-content/uploads/2018/01/wpDiscuz-Ads-Manager.png)  
  
  

A full-fledged tool-kit for advertising in comment section of your website. Separate banner and ad managment options.
---------------------------------------------------------------------------------------------------------------------

wpDiscuz Ads Manager is a light and powerful advertisement management system that allows you to add ads in comment area (after and before comment form, after and before comment list, after and before comments). You can display adverts in comment form and on comment list. There are many locations and ability to create ad banners with different displaying logics. You can attach many ads to one banner and show them randomly on top/bottom of comment form and after X parent comment on comment list. You can use simple text, HTML, iframes and ad service codes like Google Ads in advert banners…

*   Creates advertising banners on different locations of comment box:
*   Banner Location: In Comments List after each X-th parent comment
*   Banner Location: Comments Form Top
*   Banner Location: Comments Form Bottom
*   Banner Location: Comments Box Top
*   Banner Location: Comments Box Bottom
*   Allows to create ads based on Text, HTML or Google Ads like codes
*   Option to set Start and End date for each ad.
*   Option to show ad in certain banner/location
*   Option to hide/show ads for certain User Roles
*   Option to hide/show ads on certain Post Types
*   Option to exclude ads from based on Post IDs
*   Manual and Auto ad status control(active, inactive, pending)
*   Allows to create unlimited number of ad banners and attach to certain display location
*   Option to have multiple ads in one banner/location and display them randomly

Please note: this is an add-on for wpDiscuz plugin. This addon requires at least wpDiscuz 4.0.0 version.
